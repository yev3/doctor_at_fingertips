Directories:

+ The FreeRTOS-Plus/Source directory contains source code for some of the
  FreeRTOS+ components.  These subdirectories contain further readme files and
  links to documentation.

+ The FreeRTOS-Plus/Demo directory contains pre-configured demo applications
  that use the TCP/IP and file system code on a variety of platforms.

See:
http://www.freertos.org/FreeRTOS-Plus/FreeRTOS_Plus_TCP/TCP_FAT_demo_projects.html
